# Amantac Frontend

واجهة المستخدم لنظام أمانتاك لإدارة المدرسة

## التقنيات المستخدمة

- **React 18** - مكتبة UI
- **TypeScript** - للكتابة الآمنة
- **Vite** - Build tool سريع
- **Tailwind CSS** - للتصميم العصري
- **React Router** - للتنقل بين الصفحات
- **Zustand** - لإدارة الحالة
- **React Query** - لإدارة البيانات والـ API calls
- **Axios** - للاتصال بالـ Backend
- **Lucide React** - للأيقونات
- **React Hot Toast** - للإشعارات

## البدء

### التثبيت

```bash
cd frontend
npm install
```

### التطوير

```bash
npm run dev
```

التطبيق سيعمل على http://localhost:3000

### البناء للإنتاج

```bash
npm run build
```

## المتغيرات البيئية

أنشئ ملف `.env`:

```env
VITE_API_URL=http://localhost:8000/api
```

## البنية

```
src/
├── api/          # API clients
├── components/   # المكونات المشتركة
├── pages/        # صفحات التطبيق
├── store/        # State management
└── App.tsx       # المكون الرئيسي
```

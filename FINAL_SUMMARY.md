## 📝 API Endpoints

### AI Insights
- `GET /api/ai/parent-guidance/{student_id}` - Parent guidance
- `GET /api/ai/teacher/workload-reduction` - Workload tips
- `GET /api/ai/anomaly-detection` - Anomaly detection
- `GET /api/ai/smart-summary/{student_id}` - Smart summary

- `GET /api/audit/logs` - Get audit logs (Admin only)
- `GET /api/audit/logs/my` - Get my audit logs
- `GET /api/audit/stats` - Get audit statistics (Admin only)
